# 🔗 Links de Redes Sociales y Contacto

Esta es la guia rapida para actualizar los contactos reales del menu.

## 📞 Número de Teléfono

**Ubicación:** footer de `index.html`
**Formato actual:**
```html
<a href="tel:+50240917878" class="contact-link">
```

**Para cambiar:**
1. Abre `index.html`
2. Busca `<a href="tel:`
3. Reemplaza `+50240917878` y/o `+50248887300` con tus números
4. Guarda el archivo

## 📱 Redes Sociales

### Instagram
**Ubicación:** footer de `index.html`

```html
<a href="https://www.instagram.com/oasisthetraveler/" target="_blank">
```

Reemplaza `https://instagram.com` con tu perfil:
```html
<a href="https://www.instagram.com/oasisthetraveler/" target="_blank">
```

### Facebook
**Ubicación:** footer de `index.html`

```html
<a href="https://www.facebook.com/OasisTheTraveler" target="_blank">
```

Reemplaza con tu página:
```html
<a href="https://www.facebook.com/OasisTheTraveler" target="_blank">
```

### WhatsApp
**Ubicación:** footer de `index.html`

```html
<a href="https://wa.me/50240917878" target="_blank">
```

Reemplaza con tu número (formato: wa.me/codigo_pais_y_numero):
```html
<a href="https://wa.me/50240917878" target="_blank">
```

Ejemplo si tu número es +502 4091-7878:
```html
<a href="https://wa.me/50240917878" target="_blank">
```
