# 🔗 Links de Redes Sociales y Contacto

Esta es la guía rápida para actualizar los contactos en el menú.

## 📞 Número de Teléfono

**Ubicación:** `index.html` línea ~66
**Formato actual:**
```html
<a href="tel:+50278946666">
```

**Para cambiar:**
1. Abre `index.html`
2. Busca `<a href="tel:`
3. Reemplaza `+50278946666` con tu número
4. Guarda el archivo

## 📱 Redes Sociales

### Instagram
**Ubicación:** `index.html` línea ~70

```html
<a href="https://instagram.com" target="_blank">
```

Reemplaza `https://instagram.com` con tu perfil:
```html
<a href="https://instagram.com/thetravelersemucchampey" target="_blank">
```

### Facebook
**Ubicación:** `index.html` línea ~74

```html
<a href="https://facebook.com" target="_blank">
```

Reemplaza con tu página:
```html
<a href="https://facebook.com/thetravelerhotel" target="_blank">
```

### WhatsApp
**Ubicación:** `index.html` línea ~78

```html
<a href="https://whatsapp.com" target="_blank">
```

Reemplaza con tu número (formato: wa.me/codigo_pais_y_numero):
```html
<a href="https://wa.me/50278946666" target="_blank">
```

Ejemplo si tu número es +502 7894 6666:
```html
<a href="https://wa.me/50278946666" target="_blank">
```
